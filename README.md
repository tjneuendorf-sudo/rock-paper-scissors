# rock-paper-scissors

This is a command-line Rock, Paper, Scissors game built in Python.

#Features

  1) Best of 3 gameplay
  2) Tracks scores during each game
  3) Checks for a JSON save file and creates one if it does not exist
  4) Updates and saves results using JSON for persistent tracking
  5) Handles user input and validation, including edge cases such as:
      -Capital letters (examples: "Rock", "ROCK")
      -Extra spaces before or after input
  6) Prompts the player to play again after each game
